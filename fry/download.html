Download Fry

In Fry's github repository, you will find a batch interpreter and a bit buggy emacs syntax highliting mode.
The interpreter is written in Haskell, so you'll need a Haskell compiler. Any Haskell compiler would do, for example GHC. Once you have GHC, download the sources, go to the "interpreter" directory, and type
ghc --make -O2 Main -o fry
This will compile the interpreter and create an executable called "fry". Now, you can interpret a Fry file, like this:
./fry my-program.fry
Try also the -v option for more verbose statistics:
./fry -v my-program.fry
