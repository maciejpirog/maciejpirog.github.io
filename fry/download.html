Download Fry

See Fry's github repository. The Fry project consists of the following parts, put in appropriate directories:
  • interpreter - interactive interpreter written in Haskell (sources only, so you have to compile it yourself)
  • emacs-mode - a bit buggy syntax highliting mode for Fry (no support for running the interpreter within an emacs buffer yet, sorry!)
